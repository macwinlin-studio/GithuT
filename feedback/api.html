1.Feedback API

First,confirm you have a feedback account.

Then,if you want to other client can use your feedback server,please use these API:

Link Type POST Data Use Response
/login POST account,password Login code(200,401,404,500...)
/feedback POST account,password,feedback-type(bug,warn,debug,want),feedback-info Feedback code(200,401,404,500...)
/change POST account,origin,new Change Password code(200,401,404,500...)

Can use,and can not use,our feedback server don't have these API:

Link Type POST Data Use Response
/register POST email,password Register code(200,401,404,500...)
/register-organization POST email,password Register Orgazation Account code(200,401,404,500...)

You will make a json file,like this:

feedback.json:

{"org":"MacWinLin Studio CGK Team","enabled":false,"ipv4":false,"ipv6":true,"link":"http://feedback-githut.macwinlin.ml:5000"}
Name Use Value
org Feedback server organization text
enabled Feedback server enable true/false
ipv4 Have IPv4 address true/false
ipv6 Have IPv6 address true/false
link https:// domain text

And make a json file,like this:

feedback-organization.json:

{"org":"MacWinLin Studio CGK Team","enabled":false,"ipv4":false,"ipv6":true,"link":"http://feedback-organization-githut.macwinlin.ml:5000"}

Two files,then OK.

Our feedback json file is in:https://githut.macwinlin.ml/feedback.json and https://githut.macwinlin.ml/feedback-organization.json
