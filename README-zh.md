## 背景

我不喜欢使用GitHub网页进行操作。我想要使用代码操作GitHub，Git帮助了我。

但其他操作呢？所以我新建了这个项目并更新。

## 安装

请先安装[Python](https://www.python.org)

然后打开`shell.py`，然后您可以使用GithuT！
